# Fadyedit
Simple and lightweight text editor built with Qt 5.15 and C++ based on One Dark theme which supports:
1. Add, close and move tabs.
2. Read, write and save files.
3. File explorer.
4. Zoom in and zoom out.
5. Nice user interface.
